# Case-Social-Connect
DESCRIÇÃO DO TRABALHO
A plataforma Social Connect, permite que as pessoas se conectem e interajam, a partir da criação de um perfil online. A proposta da plataforma é ser versatil e aumentar a conexão entre as pessoas.

TECNOLOGIAS UTILIZADAS
Foram Utilizadas as seguintes tecnologias para a confecção do trabalho:
-HTML5
-CSS

ATUALIZAÇÕES FEITAS:

-Correção das Imagens 
-Adição das redes sociais
-Correção do rodapé
-Adição dos contatos
